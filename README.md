# KIWI'S BARK BLASTER

### *A 4-pound Pomeranian. An army of squirrels. One backyard to defend.*

**Kiwi's Bark Blaster** is a critically acclaimed*, award-winning** retro arcade experience that puts you in the tiny paws of Kiwi, a fluffy orange Pomeranian who has had ENOUGH of these squirrels.

> *Not actually critically acclaimed.
> **Has won zero awards. But it should.

## Play Now

**[https://katehostetler.github.io/kiwi-bark-blaster/](https://katehostetler.github.io/kiwi-bark-blaster/)**

Works on desktop and mobile. No install required. Zero dependencies. Just pure, unfiltered bark energy.

## Features

- **Next-Gen Pixel Art Engine** - Kiwi's magnificent plumed tail was rendered using cutting-edge `ctx.arc()` technology. Each fur tuft is individually calculated at runtime because we believe in procedural fluffiness.

- **Procedural Chiptune Soundtrack** - A 16-bar banger composed entirely from oscillator nodes. Bach could never.

- **Advanced Squirrel AI** - They move left. They move right. Sometimes they drop acorns on you. We spent literally minutes on this.

- **ZOOMIES MODE** - The feature no one asked for but everyone deserves. Kiwi loses all control, rockets across the screen at 3x speed, and auto-fires barks in a spread pattern while leaving rainbow afterimages. Just like real life.

- **Boss Battles** - Every 3 waves, a Giant Raccoon descends from the heavens to challenge Kiwi. It has red eyes and an unreasonable amount of HP. We are not sorry.

- **Power-Up System** - Golden Bone (rapid fire), Treat (extra life), Kiwi Fruit (screen clear bomb that is also a pun).

- **High Score Persistence** - Your greatest achievements are saved to localStorage, which means they'll survive until you clear your browser data during a moment of digital hygiene.

- **Mobile Touch Controls** - On-screen buttons so you can defend the backyard from the bus, the couch, or the bathroom. We don't judge.

## Controls

### Desktop
| Key | Action |
|-----|--------|
| Arrow Left / Right | Move Kiwi |
| Space | BARK (shoot) |
| Z | Activate ZOOMIES |
| Enter | Start / Restart |

### Mobile
Tap the screen to start. Use the on-screen buttons: arrows to move, BARK to shoot, Z for zoomies.

## Lore

In the beginning, there was a backyard. And in that backyard, there was peace.

Then the squirrels came.

They arrived in formation — rows upon rows of bushy-tailed invaders descending from the oak tree with military precision. They brought acorns. They brought attitude. They brought a Giant Raccoon.

Only one stood between the backyard and total squirrel domination: a 4-pound Pomeranian named Kiwi, armed with nothing but an earth-shattering bark and an unbreakable spirit.

The neighbors have filed noise complaints.

Kiwi does not care.

## System Requirements

- A web browser invented after 2015
- A device with at least one pixel
- The spiritual fortitude to face the Giant Raccoon
- Headphones recommended (your coworkers will thank you)

## Tech Stack

This entire game is a single HTML file. No frameworks. No build tools. No node_modules folder consuming 847MB of disk space. Just one file, doing everything, like a Pomeranian who believes she is a wolf.

- HTML5 Canvas
- Web Audio API
- Pure JavaScript
- localStorage (for glory)
- Vibes (for everything else)
